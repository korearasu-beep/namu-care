# 나무재가방문요양 홈페이지

## 프로젝트 개요
- 방문재가요양 서비스 홈페이지
- 모든 콘텐츠(사진, 텍스트, 연락처)를 Notion CMS에서 관리
- 사용자가 코드 수정 없이 Notion만으로 홈페이지 콘텐츠 관리 가능

## 기술 스택
- React 18 + Vite
- React Router v6 (4페이지 SPA)
- Tailwind CSS
- Notion API 연동 (모든 콘텐츠 관리)
- Vercel 배포 + Serverless Functions

## 페이지 구성
1. 메인(랜딩) — 히어로 + 서비스 + 비교표 + 선택이유 + CTA
2. 회사소개 — 센터장 인사말 + 핵심가치 + 연혁 + 팀
3. 게시판 — 공지사항 탭 + 활동앨범 탭
4. 상담신청 — 폼 + 지도 + 연락처

## Notion CMS 구조 (DB 6개)
1. 메인콘텐츠 - 히어로 텍스트, 통계, 이미지 등
2. 회사소개 - 인사말, 비전, 연혁, 팀사진
3. 공지사항 - 제목, 카테고리, 날짜, 내용
4. 활동앨범 - 제목, 날짜, 사진들
5. 사이트설정 - 전화번호, 이메일, 주소, 로고 등
6. 서비스사진 - 서비스별 사진, 설명, 태그
